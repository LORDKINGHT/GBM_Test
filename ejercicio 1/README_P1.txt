Verificador de Palíndromos

Este script de Python comprueba si una cadena de texto es un palíndromo.

Funcionalidad

El script is_palindrome.py contiene una función llamada is_palindrome(word) que toma una palabra como entrada y verifica si es un palíndromo. Un palíndromo es una palabra o frase que se lee igual hacia adelante y hacia atrás.

La función realiza los siguientes pasos:

1. Convierte la palabra a minúsculas utilizando el método lower().
2. Elimina los espacios en blanco de la palabra utilizando el método replace(" ", "").
3. Compara la palabra original con su versión invertida para determinar si es un palíndromo.
