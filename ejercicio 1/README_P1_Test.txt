PRUEBAS DE PALINDROMOS
Este es un conjunto de pruebas unitarias para verificar si una cadena de texto es un palíndromo o no.

Requisitos

1. Python 3.x
2. Biblioteca estándar de Python

Ejecución de las pruebas
1. Asegúrate de tener Python instalado en tu sistema.
2. Descarga el archivo P1.py y guárdalo en tu directorio de trabajo.
3. Descarga este archivo README.md para obtener instrucciones sobre cómo ejecutar las pruebas.
4. Abre una terminal y navega hasta el directorio donde se encuentran los archivos descargados.

Ejecuta el siguiente comando para ejecutar las pruebas:

python P1_test.py

Detalles de las pruebas
1. test_palindromo_not_spaces: Prueba si las cadenas sin espacios son reconocidas correctamente como palíndromos.
2. test_palindrome_with_spaces: Prueba si las cadenas con espacios son reconocidas correctamente como palíndromos.
3. test_not_palindrome: Prueba si las cadenas no son reconocidas correctamente como palíndromos.

Resultados Esperados
1. Cada prueba debe pasar exitosamente si la función is_palindrome() implementada en P1.py funciona correctamente.
2. Se espera que las cadenas que sean palíndromos devuelvan True, mientras que las cadenas que no lo sean devuelvan False.
